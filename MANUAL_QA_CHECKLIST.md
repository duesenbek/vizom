# Vizom MVP - Manual QA Checklist

---

## index.html QA Checklist

### Functional Tests
- [ ] Page loads without console errors
- [ ] All buttons respond to clicks
- [ ] All forms validate and submit
- [ ] All links navigate correctly
- [ ] Interactive elements work (dropdowns, toggles, modals)
- [ ] CTA button "Get Started Free" navigates to generator
- [ ] "View Templates" navigates to templates page
- [ ] Email signup form (if present) validates email format

### Visual Tests (Desktop - 1920x1080)
- [ ] Header renders correctly
- [ ] Hero section aligned
- [ ] Content sections have proper spacing
- [ ] Footer renders correctly
- [ ] No text overflow or cut-off
- [ ] Images load properly

### Visual Tests (Tablet - 768px)
- [ ] Layout adapts to tablet width
- [ ] Navigation menu works
- [ ] Text remains readable
- [ ] Buttons are tappable (min 44px)

### Visual Tests (Mobile - 375px)
- [ ] Mobile menu opens/closes
- [ ] Content stacks vertically
- [ ] No horizontal scroll
- [ ] Touch targets adequate

### Performance Tests
- [ ] Page loads in < 3 seconds
- [ ] No 404 errors in Network tab
- [ ] No render-blocking resources
- [ ] Lighthouse score > 80

### Browser Compatibility
- [ ] Works in Chrome (latest)
- [ ] Works in Safari (latest)
- [ ] Works in Firefox (latest)
